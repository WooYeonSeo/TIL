웹서버
---------
1.기능
요청을 보내고 그 응답을 보내는 게 가장 중요한 기능
그 리소스를 http 형식에 맞춰서 보내게 되고 
그 때 문제 발생하면 에러 띄움

2.요청하는 데이터
- 정적 데이터 : 이미지 html css js 파일들
- 동적 결과 : 웹서버에 프로그램을 통한 결과들

요청한 데이터에 대한 결과를 브라우저에 돌려주거
브라우저는 그 안의 파일들 html js 등을 합쳐서 해석해서 보여줌(=랜더링)

* 웹 크롤러 : 네이버나 구글에서 다른 웹사이트 정보 읽을때 쓰는 소프트웨어

3.http : 인터넷 통신 프로토콜(=규약)

4.서버 종류 : nginx, apache 
apache :  온픈소스 대부분의 os에서가능
nginx : 적은 리소스로 운영하도록 설계, 오픈소스 ** 차세대
 + 아파치가 압도적이지만 엔진엑스도 늘고있음
